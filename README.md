# 二叉树可视化分析器使用教程

你不需要启动本地服务器，也不需要安装 Node、npm、Python 或任何插件。只要有一个浏览器，就可以使用。

## 页面里每个按钮是做什么的

### 构建并分析

当你手动输入一棵新的二叉树后，点击这个按钮，页面会重新建树并刷新全部结果。

### 载入样例

如果你把输入框改乱了，或者只是想恢复到老师题目里的示例数据，点这个按钮就行。

样例内容是：

```text
[8, 4, 12, 2, 6, 10, 14, null, null, 5, 7]
```

### 重置

清空当前输入和查询内容，把页面恢复到空树状态。

### 查询节点

在“目标值”输入框里输入一个数字，比如 `7`，然后点“查询节点”，页面会告诉你：

- 这个值是否存在
- 从根节点到这个值的路径
- 它位于第几层
- 图形树里对应的节点会被高亮

## 你应该怎么输入一棵树

这个页面使用“层序输入法”。

你可以把它理解成：从上到下、从左到右，把节点一个一个写进数组里。

空节点写成 `null`。

例如：

```text
[8, 4, 12, 2, 6, 10, 14, null, null, 5, 7]
```

它表示的是这样一棵树：

```text
				8
			/   \
		 4     12
		/ \    / \
	 2   6  10 14
			/ \
		 5   7
```

## 输入时的几个注意点

1. 最外面最好带上中括号，例如 `[1, 2, 3]`。
2. 数字之间用逗号分隔。
3. 空节点写 `null`。
4. 中文逗号也能识别。
5. `NULL` 也能识别。
6. 不要把字母当成节点值输入。

## 页面会给你看什么结果

当你点击“构建并分析”以后，页面会自动给出下面这些内容：

- 前序遍历
- 中序遍历
- 后序遍历
- 层序遍历
- 非递归前序遍历
- 节点总数
- 叶子节点数量
- 树的高度
- 每层节点数
- 最大宽度
- 是否为空树
- 是否为满二叉树
- 是否为完全二叉树
- 是否为平衡二叉树
- 是否为二叉搜索树
- 图形树
- 竖状文本树

## 如果页面看起来不对，先这样处理

### 情况 1：页面能打开，但结果没显示

先点一次“载入样例”。

### 情况 2：你改过输入后报错

直接点“载入样例”，恢复到正确示例。

### 情况 3：查询没有结果

检查你输入的是不是数字，比如 `7`、`10`、`14`。

### 情况 4：想重新开始

点“重置”，然后再点“载入样例”或者自己重新输入。
